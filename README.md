## What each part covers

### Part 1 — Data Cleaning and Transformation

**Section 1 — Missing Data Handling (4 marks)**
- Dataset inspection: shape, dtypes, describe
- Missing value audit with heatmap and bar chart
- Strategy: drop high-missing columns (`weight` 97%, `medical_specialty` 49%, `payer_code` 40%); mode-impute `race` (2.2%); fill `diag_1/2/3` NaN with `-1`
- Split numerical vs categorical; verify resolution

**Section 2 — Cleaning and Transformation (3 marks)**
- Encode `readmitted` to binary (any readmission = 1)
- Plot readmission rates by gender, age, race
- Insulin impact analysis for readmitted patients (by age, gender, race)
- ICD-9 diagnosis codes → 10 disease categories (`Circulatory`, `Diabetes`, `Injury`, etc.)
- Encode 23 drug columns: any active use → 1, no use → 0
- Encode `A1Cresult` and `max_glu_serum` to integer (-99 for untested)

**Section 3 — Outlier Detection and Treatment (3 marks)**
- Visual exploration: box plots and scatter plots
- IQR method: compute Q1/Q3/IQR fences for 8 numerical columns
- Winsorisation: clip to `[Q1 − 1.5·IQR, Q3 + 1.5·IQR]` (45,358 values clipped)
- Before & after comparison plots

### Part 2 — Feature Selection (5 marks)
- Correlation heatmap of all numerical features
- Forward selection via Linear Regression (`k_features=5`): selects `number_inpatient`, `number_diagnoses`, `num_procedures`, `num_medications`, `discharge_disposition_id`
- Lasso Regression (alpha=0.01): retains 8 features, zeroes out 28

### Part 3 — Data Visualisation (5 marks)
Three numerical variables — `time_in_hospital`, `num_medications`, `num_lab_procedures` — each visualised with a histogram, bar graph, pie chart, and heatmap against the `readmitted` target.

## Key findings

- **Strongest readmission predictor:** `number_inpatient` (r = 0.23) — patients with prior inpatient visits are chronically ill and cycle through the hospital.
- **Most readmitted diagnosis category:** `Circulatory` (ICD-9 390–459) — cardiovascular complications are the leading comorbidity in diabetic patients.
- **Readmission rate by stay length:** rises from 39.6% (1 day) to 50.4% (8 days), confirming that longer stays reflect harder-to-resolve conditions.
- **Medication count effect:** readmission rates climb from 33.2% (1–5 drugs) to 48.7% (21+ drugs), showing polypharmacy as a marker of disease complexity.
- **Both selection methods agree** on the core four features: `number_inpatient`, `number_diagnoses`, `num_procedures`, `num_medications`.
